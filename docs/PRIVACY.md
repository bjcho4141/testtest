# 개인정보처리방침 (utube-shorts-jp)

> ⚠️ 본 문서는 **한국 SaaS 표준 1차 작성안**이며 시행 전 변호사 검토를 권장합니다.

최종 갱신일: 2026-05-06

## 1. 수집하는 개인정보 항목

| 구분 | 항목 | 수집 시점 |
|---|---|---|
| 필수 | 이메일, 카카오 닉네임, 카카오 프로필 이미지 (URL) | 카카오 로그인 시 |
| 필수 | 카카오 고유 ID | 카카오 로그인 시 |
| 필수 | 결제 정보 (카드 마지막 4자리, 결제일, 금액, 결제 수단) | 결제 시 |
| 필수 | 빌링키 (정기결제 시, 암호화 저장) | 정기결제 등록 시 |
| 자동 | 접속 로그, IP 주소, 브라우저 정보 | 서비스 이용 시 |
| 운영 | YouTube OAuth refresh token (회원 채널 업로드 시) | YouTube 연동 시 |
| 운영 | 회원이 입력한 한국 숏츠 URL, 원본 URL, 채널 메타데이터 | 페어 등록 시 |

## 2. 수집·이용 목적

- 회원 식별 및 인증
- 결제 처리 및 영수증 발행
- 서비스 제공 (변환 작업, YouTube 업로드)
- 부정 이용 방지·디버깅·고객 지원
- 법령상 의무 이행 (전자상거래법 등)

| 항목 | 보유 기간 | 근거 |
|---|---|---|

## 4. 제3자 제공 / 처리 위탁

서비스 운영을 위해 다음 위탁사에 일부 정보가 처리됩니다:

| 위탁사 | 위탁 항목 | 위탁 목적 | 보유 기간 |
|---|---|---|---|
| 토스페이먼츠 (NHN KCP) | 결제 정보 | 결제 처리·환불 | 위탁사 정책 |
| 카카오 | 인증 정보 | 로그인 | 위탁사 정책 |
| Supabase Inc. | 회원 정보·DB·Storage | 호스팅·데이터베이스 | 위탁 종료 시 |
| Vercel Inc. | 접속 로그·정적 자원 | 호스팅·CDN | 위탁 종료 시 |
| Google (YouTube Data API) | 채널 메타·OAuth | 영상 검색·업로드 | 회원 연동 해제 시 |
| ElevenLabs | 번역 텍스트 (이름·개인정보 미포함) | TTS 음성 합성 | 위탁사 정책 |
| Anthropic / OpenAI | 자막 텍스트 (이름·개인정보 미포함) | 번역 LLM | 위탁사 정책 |

회원은 위탁에 동의하지 않을 권리가 있으나, 동의하지 않을 경우 서비스 이용이 제한될 수 있습니다.

## 5. 정보주체의 권리

회원은 다음 권리를 행사할 수 있습니다:
- 개인정보 **열람** 요청 (`/dashboard/settings`)
- 개인정보 **정정·삭제** 요청
- 개인정보 처리 **정지** 요청
- 회원 **탈퇴** (즉시 파기)

권리 행사 방법: `/dashboard/settings` 또는 `{{EMAIL}}` 로 요청.

## 6. 안전성 확보 조치

- **암호화**: 빌링키 / OAuth refresh token은 pgsodium 또는 Supabase Vault로 envelope encryption.
- **접근 통제**: Row Level Security (RLS) 정책으로 본인 데이터만 접근 가능.
- **전송 보안**: 모든 통신은 HTTPS (TLS 1.2+).
- **로그 모니터링**: 비정상 접근·SQL injection 시도 alert.
- **시크릿 관리**: `.env`·`.mcp.json` 격리 + pre-commit gitleaks.

## 7. 자동 수집 도구 / 쿠키

서비스는 세션 유지를 위해 쿠키를 사용합니다. 회원은 브라우저 설정으로 쿠키를 차단할 수 있으나, 차단 시 로그인이 불가합니다.
