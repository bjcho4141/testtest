# 환불정책 (utube-shorts-jp)

> ⚠️ 본 문서는 **한국 SaaS 표준 1차 작성안**이며 시행 전 변호사 검토를 권장합니다.

최종 갱신일: 2026-05-06

## 1. 청약철회 (전자상거래법 §17)

회원은 결제 완료 시점부터 **7일 이내** 청약철회를 요구할 수 있습니다. 단, 디지털 콘텐츠의 특성상 다음 경우 제한됩니다 (전자상거래법 §17조 2항):
- **사용을 시작한 경우** (변환 1편 이상 진행)
- **즉시 제공된 서비스로서 사용 시 가치가 현저히 감소하는 경우**

회원은 결제 완료 시 청약철회 제한에 대해 별도 체크박스로 동의합니다.

## 2. 일반결제 (₩9,900 / 30일 1회)

| 시점 | 환불 비율 |
|---|---|
| 결제 후 사용 시작 전 (변환 0편) | **100%** |
| 변환 1편 사용 후 | 90% |
| 변환 2편 사용 후 | 80% |
| ... (1편당 10% 차감) | ... |
| 변환 10편 이상 사용 | 환불 불가 |

> 변환 1편 = `conversion_jobs.status='done'` 1회 기준.

## 3. 정기결제 (₩9,900 / 월 자동)

### 3.1 결제 직후 7일 내

- **변환 0편 사용 시**: 100% 환불 + 자동결제 즉시 중단
- **변환 1편 이상 사용 시**: 환불 불가 (단, 자동결제는 즉시 중단)

### 3.2 결제 후 7일 이후

- 잔여 일수에 따라 **일할 계산** 환불 (사용한 일수만큼 차감)
- 예: 결제 15일 후 환불 신청 → 잔여 15일 × ₩330/일 = ₩4,950 환불
- 자동결제는 즉시 중단

### 3.3 다음 결제일 전 해지 (환불 아님)

- `/dashboard/billing` → "구독 해지" 버튼
- 해지 즉시 자동결제 중단, 잔여 기간(다음 결제일까지) 정상 사용 가능
- 환불 없음 (사용권 유지)

## 4. 환불 절차

1. `/dashboard/billing` → "환불 요청" 버튼
2. 사유 입력 (선택) → 제출
3. 회사가 24시간 이내 검토 → 승인 시 토스페이먼츠 API로 환불 실행
4. **환불 처리 시간**: 카드 영업일 기준 3~4일 (토스페이먼츠 정책)
5. 환불 완료 시 이메일 알림

## 5. 환불 거부 사유

다음 경우 환불이 제한됩니다:
- 약관 위반 (이용 정지·자격 박탈된 회원)
- 부정 결제·자동화 봇 사용 적발
- 환불 정책 외 사용량 (위 표 참조)

## 6. 분쟁 해결

환불 분쟁 발생 시:
1. **1차**: `{{EMAIL}}` 로 이의 제기
2. **2차**: 한국소비자원 또는 전자거래분쟁조정위원회
3. **3차**: 서울중앙지방법원 (관할)

## 7. 환불 신청

- **온라인**: `/dashboard/billing` → "환불 요청"
- **이메일**: `{{EMAIL}}` (제목: [환불] + 회원 이메일 + 결제 ID)
